## *本demo使用koa2的一个基于nodejs的下一代web框架 + mysql 实现简单数据库增删查改 ##


#### 1、先安装依赖 ####
- npm install

#### 2、建立数据库book, 导入sql文件 ####
- 数据库名：book
- 表名：book
- 用户名：root
- 密码：root

#### 3、启动node ####
- npm run dev
- 地址栏打开： http://localhost:3000/


注：
1. 需要有mysql server环境和node环境
2. 单独运行node uploads.js 实现单和多文件上传功能
